Meet the Flameberry Engine

Flameberry uses the Vulkan API which is the modern standard for cross platform graphical applications. It provides low level control over GPU resources and memory management

Powered by Nvidia PhysX allowing real time high quality physics simulation performed on parallel threads

Immediate Mode UI supports realtime, efficient and responsive user interface elements

CSharp scripting enables writing of scripts for the game entities in an easy, fast and familiar way. It provides ease of accessibility, flexibility, and platform independence

GPU accelerated Ray Tracing support is coming soon enabling extremely realistic scene rendering and natural lighting effects
